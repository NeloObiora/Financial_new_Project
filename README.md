# Financial_new Project
## 1. Project Overview

This project investigates the relationship between financial news sentiment 
and economic indicators across countries. Data was collected from three sources:

- **CNBC Finance** — financial news headlines and dates
- **World Government Bonds** — bond yields and credit ratings per country  
- **Yahoo Finance** — most active stock prices and daily percentage changes

Three scraping tools were used — requests with BeautifulSoup, Selenium, 
and Scrapy to demonstrate familiarity with different web scraping approaches.
